# ms-filetime
=============

Easily convert Microsoft FILETIME value to GMT time.

## Installation
```
  npm install ms-filetime --save
```
## Usage
```javascript
  var MConvert = require('MSFTConvert');

  console.log(MConvert('2985901175'));
```
```
Output:  Mon, 01 Jan 1601 04:57:13 GMT
```
